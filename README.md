# Payment Failure Analysis Dashboard (Power BI)

## 📊 Overview
This project analyzes 598K+ payment transactions across ATM, POS, and Online channels to identify failure patterns, anomalies, and performance gaps.

## 🔍 Key Insights
- Overall failure rate: 16% (above industry benchmark)
- ATM failure rate: 21% (highest among channels)
- Peak-hour spikes: 10AM–1PM & 5PM–9PM
- RuPay network shows highest failure (~19.7%)
- Lucknow identified as highest-risk location (20%)

## 🛠 Tools & Technologies
- Power BI
- DAX
- Data Modeling

## 📈 Features
- Executive summary dashboard
- Failure pattern analysis
- Geographic risk segmentation
- Transaction profiling
- Anomaly detection (Mean + 2SD)
- Actionable recommendations

## 📌 Business Impact
This dashboard helps identify system bottlenecks and provides data-driven recommendations to reduce transaction failures and improve customer experience.
